# Core Municipal Inteligente - Blazor Normalized

Base frontend Blazor normalizada desde salida de Instruct UI.

## Ejecutar local

```bash
dotnet restore
dotnet build
dotnet run
```

## Docker

```bash
docker build -t core-municipal-blazor .
docker run --rm -p 8080:8080 core-municipal-blazor
```

Luego abrir http://localhost:8080.
